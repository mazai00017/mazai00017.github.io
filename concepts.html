Key Concepts

This page explains some important terms connected to HTTP and HTTPS.

Important Terms

  • Client-server model: the browser sends a request and the server sends a response.
  • Stateless protocol: HTTP does not remember previous requests on its own.
  • TCP: used by earlier HTTP versions for reliable delivery.
  • QUIC: used by HTTP/3 to improve performance.
  • TLS: secures communication for HTTPS.

Concept Summary Table

Concept Meaning
Client-server Communication between the browser and web server
Stateless No memory of earlier requests unless extra tools are used
TLS Encryption that protects HTTPS traffic
QUIC Newer transport method used in HTTP/3
